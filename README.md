# react-native-directory-structure-sample
React Nativeのディレクトリ構造をテンプレートから学ぶ

# How to run

```
$ git clone https://github.com/tetsushi-ito/react-native-directory-structure-sample.git
$ cd react-native-directory-structure-sample
$ npm install
$ npm start
```
